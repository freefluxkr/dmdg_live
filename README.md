# 🍊 당목담글 - 마음을 이어주는 따스한 목소리 릴레이

> **"지친 하루 끝에 건네는 따뜻한 소리 마중물과 일본어 공감 낭독"**
> 
> 당목담글은 사용자가 '행복한가'의 아름다운 글귀(시, 수필, 오늘의 말)를 감상하고, 자신의 따뜻한 목소리로 녹음하여 마음을 이어붙이는 **온디바이스(On-device) 무료 오프라인 PWA 웹 애플리케이션**입니다.
> 재생 시, 원래 녹음된 소리와 함께 따뜻하고 다정한 **일본어 구어체 공감 멘트가 동시에 낭독(TTS)**되어 국경을 넘어 온기를 전합니다.

---

## 🎨 주요 핵심 기능

1. **Firebase Firestore 연동 감정 필터링**
   * '행복한가'의 일상스토리/문화생활정보 카테고리에서 가져온 글귀 데이터가 Firebase Firestore(`dmdg-pwa`)에 상시 저장되어 있습니다.
   * 사용자가 **[위로, 설렘, 평온]** 세 가지 감정 탭을 터치하면 해당 감정에 매칭되는 오늘의 마중물 편지가 Firebase로부터 실시간 로드되어 교체됩니다.

2. **온디바이스 오프라인 마이크 녹음 & 재생**
   * 모바일 기기(Android Chrome, iOS Safari) 및 데스크톱 환경의 마이크 장치 호환성을 극대화하여 다양한 mimeType(`audio/webm`, `audio/mp4`, `audio/aac` 등)을 기기별로 선별 적용합니다.
   * 녹음된 음성 파일은 기기 내부 저장소인 **IndexedDB(Dexie.js)**에 안전하게 보관되어 네트워크가 끊겨도 자유롭게 재생할 수 있습니다.

3. **일본어 공감 멘트 동시 TTS 낭독**
   * 음성 릴레이 카드를 재생하면, 사용자가 녹음했던 원본 목소리(한국어)가 먼저 흘러나옵니다.
   * 이어서 Web Speech API(TTS) 엔진을 통해 각 카드에 지정된 따뜻하고 귀여운 **일본어 구어체 공감 멘트**가 다정한 목소리로 흘러나와 감성을 자극합니다.
   * `0.8x`, `1.0x`, `1.2x`, `1.5x` 등 사용자가 조절한 속도에 맞춰 음성과 일어 낭독 모두 템포가 조절됩니다.

4. **마음 오디오북 연속 재생 기능 (🎧)**
   * 등록된 모든 감정 카드들을 한 번의 탭으로 묶어서 연속 재생해 줍니다.
   * 한국어 녹음 재생 ➔ 일본어 TTS 낭독 ➔ 다음 트랙으로의 자연스러운 화면 슬라이딩과 오토 플레이를 지원하는 완성도 높은 감성 플레이어 창이 동작합니다.

5. **완벽한 오프라인 PWA (Progressive Web App)**
   * `sw.js` (서비스 워커)와 캐시 무력화 패턴을 탑재하여 인터넷이 끊겨도 앱을 홈 화면에 추가하고 완벽히 독립적으로 구동할 수 있습니다.
   * **데이터 백업 및 복원**: 소장하고 싶은 내 음성 기록 리스트를 `.json` 파일로 안전하게 백업 및 파일 탐색기 연동을 통해 복원할 수 있습니다. (카메라/캠코더 등의 불필요한 시스템 접근 필터링 완료)

---

## 🛠 기술 스택 (Tech Stack)

* **Frontend**: Pure HTML5, Tailwind CSS, Lucide Icons, Google Fonts (Gowun Batang, Comfortaa, Gaegu, Pretendard)
* **Local Database**: [Dexie.js](https://dexie.org/) (IndexedDB wrapper) - 버전 3 스키마 구성
* **Backend Database**: Firebase Firestore v10.8.0 (Compat SDK)
* **Deployment**: GitHub Pages (배포 완료)

---

## 💾 로컬 데이터베이스 (Dexie.js) 스키마

* `personas`: 각 감정 큐레이터의 정보 (하루, 사쿠라, 렌)
* `primers`: Firebase Firestore로부터 캐시 또는 로컬에 동기화한 마중물 편지 내용들
* `relays`: 사용자가 생성한 오디오 녹음 및 멘트 기록들 (`audioBlob` 바이너리 타입 포함)
