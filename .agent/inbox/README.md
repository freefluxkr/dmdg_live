# 📁 당목담글 AI Agents — 공유 인박스

사장님이 각 직원에게 파일을 전달할 수 있는 공유 폴더입니다.

## 📌 폴더 구조

```
inbox/
├── all/          ← 전체 팀 공유 파일 (모든 직원 열람)
├── demis/        ← 👔 데미스 CEO 전용
├── peggy/        ← 📋 페기 비서 전용
├── mustafa/      ← 🔬 무스타파 데이터/리서치 전용
├── jennifer/     ← 🎨 제니퍼 디자이너 전용
├── craig/        ← 💻 크레이그 개발자 전용
├── haruki/       ← ✍️ 하루키 스토리작가 전용
├── zimmer/       ← 🎧 짐머 사운드디렉터 전용
├── satya/        ← ⚖️ 사티아 비판/조직운영 전용
└── sherlock/     ← 🔍 셜록 탐정/리서처 전용
```

## 📲 텔레그램으로 파일 전달하는 방법

1. 텔레그램 봇에 파일 전송
2. 캡션에 `@수신자이름` 또는 `/단축키` 입력
   - 예: `@데미스 검토 부탁드립니다`
   - 예: `@크레이그 참고 문서입니다`
3. 파일이 해당 직원 인박스에 자동 저장됨

## 📂 지원 파일 형식

PDF, DOC, DOCX, XLS, XLSX, PPT, PPTX,  
JPG, PNG, GIF, MP3, MP4, TXT, MD, ZIP 등 모든 파일

## 🤖 텔레그램 봇 명령어

| 명령어 | 기능 |
|--------|------|
| `/start` | 봇 시작 & 사용법 |
| `/team` | 전체 팀 프로필 조회 |
| `/inbox` | 인박스 파일 현황 |
| `/exec git status` | 시스템 명령 실행 |
| `@데미스 내용` | 데미스에게 직접 메시지 |
