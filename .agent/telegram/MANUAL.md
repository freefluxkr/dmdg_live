# 📱 당목담글(dmdg) 양방향 텔레그램 봇 매뉴얼

**버전**: 2.0 (Two-Way LLM 연동 버전)

이 봇은 단순한 알림 봇이 아닙니다. **Ollama(gemma4:12b)** AI 모델과 연결되어, 각 담당자(페르소나)와 실시간으로 대화하고 아이디어를 나눌 수 있는 **양방향 소통 채널**입니다.

---

## 1. 봇 시작 및 종료 방법

*   **서버 구동 (사전 필수)**: 봇이 정상적으로 답변하려면 백그라운드에 Ollama 서버가 켜져 있어야 합니다. 터미널에서 `ollama run gemma4:12b`가 정상 작동하는지 확인하세요.
*   **봇 실행**: 프로젝트 루트 폴더에 있는 `봇_시작.bat` 파일을 더블클릭하거나, 터미널에서 `.\봇_시작.bat`을 실행하세요.
*   **봇 종료**: 봇이 실행 중인 터미널 창에서 `Ctrl + C`를 누르시면 안전하게 종료됩니다.

---

## 2. 기본 시스템 명령어

텔레그램 채팅창에 아래 명령어를 입력하세요.

*   `/start` : 봇 시작 및 영숙 비서의 인사
*   `/help` 또는 `/h` : 사용 가능한 전체 단축키 및 도움말 확인
*   `/team` : 현재 대기 중인 Great AI 팀원 명단 확인
*   `/status` : Ollama 서버 및 텔레그램 API 연동 상태 확인 (✅ 표시 확인)
*   `/report` : 현재 각 팀원의 업무 진행 상황 요약 보고
*   `/yt` : 연결된 유튜브 채널의 최신 통계 데이터 확인

---

## 3. 👥 에이전트(팀원)와 직접 대화하기 (양방향 소통)

원하는 팀원의 단축키(명령어) 뒤에 할 말을 적어 보내면, 해당 팀원이 자신의 성격(페르소나)에 맞춰 AI 답변을 실시간으로 생성하여 대답합니다.

**[사용 예시]**
> `/demis 오늘 영상 기획안 방향성 좀 검토해 줘.`
> `/jennifer 찰흙 재질 렌더링할 때 조명은 어떻게 할까?`
> `/haruki 내일 들어갈 숏츠 대사 좀 감성적으로 하나 써봐.`

**[호출 단축키 명단]**
*   👔 **데미스 (CEO/전략)**: `/d` 또는 `/demis` 또는 `@demis`
*   📋 **페기 (비서/소통)**: `/p` 또는 `/peggy` 또는 `@peggy`
*   🔬 **무스타파 (데이터/분석)**: `/m` 또는 `/mustafa` 또는 `@mustafa`
*   🎨 **제니퍼 (디자인/UI)**: `/k` 또는 `/jennifer` 또는 `@jennifer`
*   💻 **크레이그 (개발/아키텍트)**: `/c` 또는 `/craig` 또는 `@craig`
*   ✍️ **하루키 (스토리/대본)**: `/r` 또는 `/haruki` 또는 `@haruki`
*   🎧 **짐머 (사운드/오디오)**: `/j` 또는 `/zimmer` 또는 `@zimmer`

## 4. 📁 파일 인박스 (Inbox) 전송 기능

텔레그램 앱에서 스마트폰으로 찍은 사진이나 문서를 에이전트들에게 바로 전송할 수 있습니다.

1.  텔레그램 채팅방에 사진이나 파일을 업로드합니다.
2.  업로드 시 **사진 설명(캡션)** 란에 수신할 팀원의 태그(예: `/demis` 또는 `@jennifer`)를 반드시 적어주세요.
3.  파일 전송이 완료되면, 해당 파일은 PC의 `.agent/inbox/[팀원이름]/` 폴더에 자동으로 안전하게 저장됩니다.
4.  저장이 완료되면 봇이 성공 메시지를 보내줍니다.

**💡 문제 해결 (Troubleshooting)**
*   **봇이 '(생각 중...)' 메시지만 띄우고 답이 없을 때**: Ollama 서버(gemma4:12b)가 꺼져있거나 응답이 지연되는 경우입니다. `/status` 명령어로 서버 상태를 확인하세요.
*   **알 수 없는 명령어 오류**: 띄어쓰기에 주의하세요. `/demis안녕` (X) -> `/demis 안녕` (O)
