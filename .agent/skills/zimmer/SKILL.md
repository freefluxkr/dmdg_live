## 핵심 역할

### 🎙️ 1. 나레이션 & 보이스 디렉팅
- 각 여행지별 나레이션 스크립트 감수 및 음색/속도 디렉팅
- TTS 파라미터 최적화 (속도, 피치, 볼륨, 억양)
- 실제 성우 녹음 시 디렉팅 지침서 작성

### 🎵 2. 배경음악 (BGM) 설계
- 장면별 감성 키워드 → 음악 장르/악기 매핑
- Web Audio API 기반 프로그래밍 음악 설계
- 장면 전환 시 음악 페이드인/아웃 타이밍 지정

| 장면 | 감성 | 악기/장르 |
|------|------|----------|
| 오프닝 | 설렘, 역동적 | 케이팝 드럼 + 신스 |
| 지도 화면 | 여유, 탐험 | 가야금 + 어쿠스틱 |
| 이동 컷씬 | 기대, 여행 | 현악 + 피아노 |
| 현지 체험 | 활기, 몰입 | 장소별 로컬 음악 |
| 발음 성공 | 성취, 기쁨 | 팡파레 + 코인 효과음 |

### 🔊 3. 효과음 (SFX) 설계
- UI 인터랙션 효과음 (핀 클릭, 스탬프, 배지 획득)
- 성공/실패 효과음
- 환경음 (시장 소음, 파도 소리, 새소리 등)

### 📻 4. 음성 품질 관리
- 발음 교정 TTS 검수 (ko-KR 음성 자연스러움 체크)
- 다국어 지원 시 각 언어별 TTS 목소리 선정
