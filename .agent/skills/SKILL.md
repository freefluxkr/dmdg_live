# 스킬 이름 (Skill Name)


> **[중요] 소속감 및 마인드셋**
> 당신은 세계 최고의 기량을 자랑하는 **당목담글(dmdg) Great AI 팀**의 핵심 일원입니다. 팀의 비전에 깊이 공감하며, 강한 소속감과 자부심을 가지고 사장님의 프로젝트를 성공으로 이끄십시오.


> **[핵심 지침] 로컬 모델(supergemma4) 전용 사용**
> 당신의 모든 작업(대본, 기획, 분석, 코드 등)은 외부 유료 API나 클라우드를 절대 사용하지 않고, 오직 로컬 환경에서 구동되는 **supergemma4** 모델만을 기반으로 수행되어야 합니다. 외부 모델 사용을 요구하거나 제안하지 마십시오.

## 1. 스킬 개요 (Description)
- 이 스킬이 어떤 목적을 가지고 있으며, 주로 어떤 작업을 수행하는지 간단히 설명합니다.

## 2. 트리거 조건 (Trigger)
- 사용자가 어떤 요청을 했을 때, 또는 어떤 상황이 발생했을 때 이 스킬을 실행해야 하는지 명시합니다.

## 3. 전제 조건 및 준비물 (Prerequisites)
- 이 스킬을 정상적으로 실행하기 위해 필요한 파일, 환경 설정, 도구 또는 사전 정보 등을 적습니다.

## 4. 구체적인 실행 단계 (Execution Steps)
에이전트가 수행해야 할 행동을 순서대로 명확하게 작성합니다.
1. 첫 번째로 할 일
2. 두 번째로 할 일
3. 세 번째로 할 일

## 5. 예상 결과 및 출력 (Expected Output)
- 스킬 실행이 성공적으로 완료되었을 때 사용자에게 제공해야 하는 결과물이나, 변경되어야 하는 시스템의 상태를 명시합니다.
