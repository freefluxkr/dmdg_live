### 1. 프론트엔드 UI/UX 대규모 개편 및 최적화

- **오디오 팝업(모달) UI 도입**: 평소에는 화면을 넓게 쓰고, 사용자가 '녹음'을 하거나 '다른 사람의 릴레이'를 재생할 때만 중앙에 예쁜 오디오 팝업이 뜨도록 구현했습니다.

### 2. 다국어 및 번역 기능 최적화
- **일본어 배경음(TTS) 삭제**: 기존 한국어 재생과 겹쳐서 기괴하게 들리던 일본어 합성음 엔진 부분을 과감하게 삭제하여 원본 목소리의 감동에 집중하도록 했습니다.
- **지원 언어 정상화**: `fr`(프랑스어), `de`(독일어) 선택 시 `i18n` 사전에 데이터가 누락되어 언어가 변경되지 않던 치명적 오류를 해결했습니다.

### 3. 런타임 JS 버그 및 캐시 동기화 해결 (트러블슈팅 완료)
- **자바스크립트 초기화 Crash 완전 해결 (Root Cause 제거)**: 기존 물방울 캔버스를 HTML에서 삭제한 뒤에도, 자바스크립트에 남아있던 `canvas.getContext('2d')` 코드가 앱 전체를 정지시키던(TDZ 연쇄 에러) 치명적 문제를 완전히 뿌리 뽑았습니다. (코드 80여 줄 삭제)
  - 이 조치로 인해 작동을 멈췄던 **번역 박스 출력, 글로벌 온기 대시보드, 녹음 기능**이 모두 정상화되었습니다.
- **오디오북 알림창(Toast) 위치 최적화**: 스마트폰 환경에서 잘리지 않고 명확히 보이도록 화면 하단(`bottom-6`)에서 상단(`top-20`)으로 위치를 변경했습니다.
- **Service Worker 강제 갱신 (`v14`)**: 브라우저가 과거의 고장난 코드를 끈질기게 캐싱하는 문제를 해결하기 위해, 서비스 워커의 버전을 `v14`로 강제 펌핑하여 배포했습니다.

### 4. Git 보안 룰렛 및 저장소 최적화
