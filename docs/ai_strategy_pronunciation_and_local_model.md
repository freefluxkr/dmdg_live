## 🎤 Gemini Live API — 발음 평가 활용 분석

### 📌 개요
Gemini 2.5 Flash의 오디오 멀티모달 기능을 활용하여  
한국어 발음을 자연어로 코칭하는 유료 프리미엄 기능을 제공한다.

### ✅ 장점

| 항목 | 내용 |
|------|------|
| **한국어 지원** | ✅ 완벽 지원 |
| **피드백 품질** | 🌟 자연어로 상세 설명 ("ㄱ 받침이 약해요", "연음이 자연스러워요") |
| **유연성** | ✅ 프롬프트 커스텀으로 감성 톤 조절 가능 |
| **무료 티어** | ✅ Google AI Studio 무료 개발 테스트 |
| **비용** | ✅ 전용 STT API 대비 저렴 |

### ⚠️ 주의사항

| 리스크 | 설명 |
|--------|------|
| **전용 점수 없음** | Azure처럼 `accuracy: 87점` 딱 떨어지는 점수 없음 → 프롬프트 설계 중요 |
| **일관성** | LLM 특성상 같은 발음에 매번 조금 다른 피드백 가능 |
| **Gemini 2.0 Flash 종료** | ⚠️ **2026년 6월 1일 서비스 종료** → 반드시 **Gemini 2.5 Flash** 사용! |

## 💵 토큰 비용 계산 (Gemini 2.5 Flash 기준)

```
오디오 변환율: 25 토큰 / 초
1회 발음 평가 = 약 10초 분량

Input:  10초 × 25토큰 = 250 토큰
        250 × ($0.30 / 1,000,000) = $0.000075

Output: 피드백 텍스트 ~200 토큰
        200 × ($2.50 / 1,000,000) = $0.000500

🎯 1회 총 비용: ≈ $0.000575 (약 0.08원!)
```

### 📊 API 원가 비교표

| API | 1회 비용 | 30회 원가 | 발음 전용 | 피드백 품질 |
|-----|---------|---------|---------|-----------|
| **Azure Pronunciation** | $0.001 | $0.030 | ✅ 전용 | 점수 위주 |
| **Naver CLOVA Speech** | ~$0.002 | $0.060 | ✅ 전용 | 점수 위주 |
| **🔥 Gemini 2.5 Flash** | **$0.00058** | **$0.017** | ❌ 범용 | **자연어 풍부** ✅ |

> 💡 **Gemini가 Azure보다 1.7배 더 저렴**하고, 피드백이 훨씬 풍부함  
> 당목담글의 따뜻한 감성에는 **Gemini 자연어 피드백**이 훨씬 잘 어울림!

```
사용자 브라우저 → index.html 내 API KEY → Gemini API
```

### ✅ 안전한 구조 (Firebase Cloud Function 프록시)
```
사용자 브라우저
    ↓ (Firebase Auth 토큰만 전송)
Firebase Cloud Function  ← API KEY 여기에만 보관!
    ↓ (서버에서 안전하게 호출)
Gemini 2.5 Flash API
    ↓
결과 반환
```

### 🛡️ 보안 레이어

| 보안 레이어 | 방법 | 역할 |
|-----------|------|------|
| **API KEY 은닉** | Firebase Function 환경변수 | 절대 노출 안 됨 |
| **사용자 인증** | Firebase Auth JWT 토큰 | 비로그인 차단 |
| **크레딧 검증** | Firestore 잔여 횟수 확인 | 무한 사용 차단 |
| **도메인 제한** | CORS 설정 | 외부 도메인 차단 |
| **속도 제한** | Function Rate Limit | DDoS 방어 |

## 🤖 "로컬 모델" 2가지 시나리오 분석

---

### 📱 시나리오 A: 앱(당목담글) 내 AI — **Gemini 2.5 Flash API 유지 (확정)**

| 항목 | 현실 |
|------|------|
| 로컬 모델 가능 여부 | WebLLM 기술적 가능 |
| 필요 사양 | 최신 스마트폰 + WebGPU 지원 |
| 모델 크기 | 최소 1~4GB 다운로드 필요 |
| 발음 평가 적합성 | ❌ 텍스트만 가능, **오디오 처리 불가** |
| **결론** | ⚠️ **발음 평가 불가 → Gemini 2.5 Flash API 유지** |

> ✅ **앱 내 발음 평가는 Gemini 2.5 Flash API를 Firebase 서버 프록시로 안전하게 사용한다.**

---

### 💬 시나리오 B: 개발 대화 — **로컬 Ollama 활용 (확정)**

현재 설치된 Ollama 모델 현황 (`ollama list` 2026-06-25 기준):

| 모델 | 크기 | 한국어 | 추천 용도 |
|------|------|--------|---------|
| `gemma4:12b` | 7.6GB | ⭐⭐⭐⭐⭐ | **전략 토론, 아이디어 브레인스토밍** |
| `gemma4:e2b` | 7.2GB | ⭐⭐⭐⭐ | 빠른 답변 |
